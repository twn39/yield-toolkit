# Yield-toolkit

Yield-toolkit is a nodejs library for generate the templates.

## Installation

npm install command:

```shell
npm i -g yield-toolkit
```

yarn install command

```shell
yarn add yield-toolkit --global
```

## Usage

Just simple command:

```shell
~ yield
```

Use arrow to select right option, yield will do the rest.
